1. To copy repository on local machine
	git clone https://github.com/Karanjot1506/fundamentals-of-python.git
2. Move to the repository folder 
	cd fundamentals-of-python
3. Check status
	git status

4. Add File to save to remote server
	git add <file name>

5. Commit command to attach message with the added file
	git commit -m "Message of the file added to refer it in future."
6. To push code to remote server

	git push

7. git pull command is used to fetch the updated data from git server.
	
	git pull
